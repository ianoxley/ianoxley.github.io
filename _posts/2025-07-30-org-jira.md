[org-jira](https://github.com/ahungry/org-jira) is an Emacs package for using Jira in Org Mode. Here's how I set it up in Doom Emacs:

1. Add `org-jira` to my `~/.config/doom/packages.el` file:
```elisp
(package! org-jira)
```
2. Create a <abbr title="Personal Access Token">PAT</abbr> in Jira
3. Store my Jira PAT in my `~/.authinfo` file
4. Configure `org-jira` in my `~/.config/doom/config.el` file:
```elisp
(use-package! org-jira
  :config
  (setq jiralib-url "<my-jira-url>")
  (setq jiralib-token
        (cons "Authorization"
              (concat "Bearer " (auth-source-pick-first-password
                                 :host "<my-jira-url>")))))
```
